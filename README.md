# Cubes

Dependencies

- glad
- glfw
- glm
- SOIL2

## Keys

`WASD` to move. Mouse to look around. `R` to toggle wireframe rendering. `Esc` to close. `Shift`/`Ctrl` to move the camera up or down (in the y axis).
